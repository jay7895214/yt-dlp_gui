# yt-dlp_gui v1.3.0

基於 Python 和 Tkinter 開發的 YouTube 與 Podcast 萬用下載器圖形介面 (GUI)。
本程式作為 `yt-dlp` 和 `ffmpeg` 的圖形化外殼，旨在提供更直觀的下載體驗。

## 核心功能
- 支援批次下載
- 支援播放清單與單一影片解析
- 提供影片格式、解析度與容器格式 (MP4/WebM) 偏好選擇
- 支援影片時間裁切與進階影片分割功能（依時間或數量）
- 支援字幕下載 (手動上傳與自動產生)，可自訂語言與字幕格式
- 支援**單獨下載字幕** (不下載影片)
- 支援將字幕與中繼資料 (Metadata) 嵌入影片檔案中
- 內建 `yt-dlp`、`ffmpeg` 與 `ffprobe` 自動更新工具
- 提供進度條、詳細執行日誌，並會**自動保存使用者的介面設定**

## 系統需求
- Python 3.x
- `yt-dlp`, `ffmpeg`, `ffprobe` (若無安裝，可直接使用程式內的「工具 -> 檢查並更新組件」自動下載)

## 使用方式
執行以下指令啟動應用程式：
```bash
python yt-dlp.py
```

## 下載行為與邏輯詳解

本程式在不同的介面提供了不同的下載按鈕，它們套用設定的邏輯如下：

| 觸發場景 | 按鈕位置 | 影片下載行為 | 字幕下載行為 |
|---------|----------|-------------|-------------|
| **直接下載** | 主視窗底端 (`⬇️ 直接下載`) | ✅ 套用**主視窗**的影片與容器設定 | ✅ 套用**主視窗**的字幕設定 |
| **快速下載選取項目** | 播放清單解析視窗 (`⬇️ 快速下載選取項目`) | ✅ 套用**主視窗**的影片與容器設定 | ✅ 套用**主視窗**的字幕設定 |
| **下載選取格式** | 單一影片詳情視窗 ➔ `📹 影片格式` 分頁 | ✅ 僅下載您所選取的格式 (忽略主視窗設定) | ❌ **不處理** (不會下載任何字幕) |
| **僅下載選取字幕** | 單一影片詳情視窗 ➔ `📝 字幕` 分頁 | ❌ **略過下載影片** (`--skip-download`) | ✅ 僅下載您所選取的字幕 (套用主視窗的字幕格式設定) |

> **提示**：如果您希望同時下載「特定的影片格式」與「特定的字幕」，目前請先在詳情視窗中確認所需的格式與語言後，回到主視窗設定好參數，再點擊「直接下載」。
